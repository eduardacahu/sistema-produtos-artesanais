# Sistema de Produtos Artesanais

Projeto desenvolvido para a disciplina de Desenvolvimento Full Stack com o objetivo de criar um sistema de cadastro e gerenciamento de produtos artesanais utilizando HTML, CSS, JavaScript, PHP e MySQL.

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Edição de produtos
- Exclusão de produtos
- Integração com banco de dados MySQL
- Atualização dinâmica da tabela utilizando JavaScript e Fetch API

## Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- PHP
- MySQL
- Git e GitHub

Campos:
